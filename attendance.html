Attendance Reminder

Automates detection of incomplete attendance and sends verification emails.

Problem

HR receives raw attendance exports with inconsistent formatting. Missing Time In/Out entries required manual follow-up which created delays and errors.

Solution

  1. Script reads latest Excel export (monthly file).
  2. Pivots the raw block format into one-row-per-person-per-day.
  3. Applies rules to detect missing Time In/Out and flags mis_tap rows.
  4. Generates a summary Excel and (optionally) sends templated emails for manager approval.
