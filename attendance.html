Attendance Reminder

Detect missing attendance and automate reminders

Overview

This project consumes attendance exports (Excel), pivots them into usable shape, detects when “Time In” or “Time Out” is missing, and optionally sends templated emails. It helps HR reduce manual work and improve compliance.

How It Works

  1. Read the latest attendance file exported from the system.
  2. Process the raw format into rows with GT ID, Name, Department, Date, Time In, Time Out.
  3. Identify records that have missing either In or Out time.
  4. Generate summary Excel and send reminders (if enabled).
