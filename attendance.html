Attendance Reminder

Google Apps Script + Gmail automation to remind staff about missing time logs.

Overview

This project automatically checks employee time logs in a Google Sheet. If a log is missing, the system sends a polite reminder email via Gmail.

How It Works

  1. Google Sheet stores daily attendance logs.
  2. A Google Apps Script scans the sheet every evening.
  3. Missing entries trigger an automated Gmail reminder.
  4. HR dashboard updates with compliance rates.
