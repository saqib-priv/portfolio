Attendance Reminder

Automated alerts for incomplete attendance

📌 Overview

This tool checks staff attendance in real-time and sends automated reminders when employees forget to mark their presence. Built with Google Apps Script, it integrates directly with Google Sheets and Gmail.

⚙️ How It Works

  1. Manager updates staff list in Google Sheets.
  2. Script runs daily at set time.
  3. If attendance is missing → Email alert is sent automatically.

💻 Code Example


// Google Apps Script
function sendAttendanceReminder() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Attendance");
  const data = sheet.getDataRange().getValues();
  data.forEach((row, i) => {
    if (i === 0) return; // skip header
    if (!row[2]) { // if attendance is empty
      MailApp.sendEmail(row[1], "Attendance Missing", "Please mark your attendance today.");
    }
  });
}
