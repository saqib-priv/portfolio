Data Pivot Tool

Python utility that transforms exported attendance into usable reports.

What it does

Pivots raw exported rows into a “one-row-per-person-per-day” structure, fills missing values according to rules, and writes an Excel with sheets "All Staff" and "mis_tap" for follow-up.
