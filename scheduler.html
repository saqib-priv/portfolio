Saturday Scheduler

Manager UI (Apps Script) for marking staff ON/OFF and tracking risks.

Overview

Managers enter their email to load their department roster. The UI generates current-month Saturdays, allows marking ON/OFF per employee per date, and saves responses back to a "Responses" sheet.

Important parts

function getDataForUI(managerEmail) {
  const { employees, department } = getEmployees(managerEmail);
  const saturdays = getCurrentMonthSaturdays();
  return {
    employees,
    saturdays: saturdays.map(d => Utilities.formatDate(d, Session.getScriptTimeZone(), 'dd-MMM-yyyy')),
    month: saturdays.length ? Utilities.formatDate(saturdays[0], Session.getScriptTimeZone(), 'MMMM') : '',
    department
  };
}

saveResponses(managerEmail, responses) appends choices to the Responses sheet.

UX Notes

  • Per-date OFF counters to highlight understaffing risk.
  • Simple select controls for quick marking.
  • Designed to run as a Sheets sidebar or Web App.
