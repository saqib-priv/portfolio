Saturday Scheduler

Manager-facing scheduler built with Google Apps Script and a lightweight UI.

Overview

Managers enter their email to load department roster, mark staff ON/OFF for each Saturday, and save responses to a 'Responses' sheet. The UI shows per-date OFF counters with thresholds that highlight potential understaffing.

Important functions (Apps Script)

function getDataForUI(managerEmail) {
  const { employees, department } = getEmployees(managerEmail);
  const saturdays = getCurrentMonthSaturdays();
  return {
    employees,
    saturdays: saturdays.map(d => Utilities.formatDate(d, Session.getScriptTimeZone(), 'dd-MMM-yyyy')),
    month: saturdays.length ? Utilities.formatDate(saturdays[0], Session.getScriptTimeZone(), 'MMMM') : '',
    department
  };
}

saveResponses(managerEmail, responses) appends manager choices to the Responses sheet.

UX notes

  • Simple, responsive UI designed to be embedded in Google Sheets sidebar or deployed as Web App.
  • Threshold rules color OFF counts red to prompt reassignments.
  • Works offline in the sheet; data saved to the spreadsheet for reporting.
