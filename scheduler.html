Saturday Work Scheduler

Manager UI (Google Apps Script + Sheets) for planning Saturdays and auto-saving responses.

Overview

Interactive UI built in Apps Script that loads employees for the manager, generates current-month Saturdays, and lets managers mark ON/OFF per employee per Saturday. Saves results to a 'Responses' sheet.

Key Apps Script functions

// getDataForUI: returns employees + current month saturdays
function getDataForUI(managerEmail) {
  const { employees, department } = getEmployees(managerEmail);
  const saturdays = getCurrentMonthSaturdays();
  return {
    employees,
    saturdays: saturdays.map(d => Utilities.formatDate(d, Session.getScriptTimeZone(), 'dd-MMM-yyyy')),
    month: saturdays.length ? Utilities.formatDate(saturdays[0], Session.getScriptTimeZone(), 'MMMM') : '',
    department
  };
}

saveResponses writes manager decisions back to the "Responses" sheet.

UX notes

  • Shows per-date OFF counters that color red when too many people are offline.
  • Manager email entry filters to their department automatically.
  • Works directly inside the sheet (no external hosting required).
